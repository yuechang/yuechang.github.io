java.io.File中mkdir和mkdirs的差别

发表于 Nov 17 2016 | 阅读次数:

mkdirs()可以成功建立多级文件夹， mkdir()只能成功建立一级的文件夹，多层目录就不能创建成功

程序运行将抛出：java.io.IOException: 系统找不到指定的路径。因为mkdir()方法没有成功创建多层目录。

建议使用：mkdirs()方法
