Java webSocket 网页聊天室

发表于 2016-07-14

webSocket简介

webSocket是ajax轮询与long pull方式的改进。ajax轮询：每隔一段时间请求一次，不管有没有新信息；long pull：每次请求之后，等到有新信息才会返回；webSocket：客户端发送一次请求之后，只要有新信息，服务器就会主动把数据推送过去。
